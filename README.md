# Shell Project

## Functionality

  - This is a basic shell implementation that can run commands found in /usr/bin 
  - A few built-in commands were implemented such as cd, help, exit, and echo
  - File i/o redirection
  - piping between processes 

## Build and Run 
  
Compile the Project
  
``` Makefile
make
```

Run the program

``` Bash 
bin/myshell
```

## Testing the shell

  ``` SHELL 
ls
```
